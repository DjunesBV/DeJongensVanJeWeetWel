# SEO-project — De Jongens Van Je Weet Wel

> **Begin hier:** lees `06-strategie-handleiding.md` voor de workflow.
> Wil je alleen de site-structuur zien? Open `SITEMAP-LEESBAAR.md`.
> Plant je sprint? Open `08-content-calendar.md`.

## Wat is dit?

SEO-strategie voor dejongensvanjeweetwel.nl met **81 nieuwe pagina's** —
allemaal onderbouwd met SEMrush zoekvolume + moeilijkheid, plus
concurrentieanalyse en content-briefings.

## Snel overzicht — wat er is gevalideerd

| Metriek | Aantal |
| --- | --- |
| URLs in sitemap | 93 (81 nieuw + 12 bestaand) |
| KWO via SEMrush (seeds) | 70+ |
| Concurrentieanalyses | 5 silo's volledig |
| Content-briefings — kort | 10 stad/silo bestanden met 80+ pagina-briefings |
| Content-briefings — volledig uitgewerkt | 10 quick wins in 8 losse files |

## Volgende stap voor de schrijver

1. Lees `06-strategie-handleiding.md`
2. Open `01-sitemap-v2.csv` — sorteer op `prioriteit` (P0) en `kd` (oplopend)
3. Pak de eerste pagina, vind de briefing, schrijf en publiceer
4. Update status in `01-sitemap-v2.csv`
